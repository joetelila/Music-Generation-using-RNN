To execute:

on terminal: 
  1. python3 mainAll.py
  2. chat predicate will be queried automatically, you can start to type
  3. when user collected emotions and quit, Anna bot will be activated 
  4: (for now) switch to telegram interface
  
Libraries to install:
  1. pip install pipswip
 
Generated files:
  1. emotions.txt which contains a list of all emotions in {happy,sad,disgust,surprise,angry}
